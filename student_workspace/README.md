# Student Workspace 🏭

This is where you'll build your Flask application during the workshop.

## Getting Started

1. **Open terminal in this folder**

2. **Run checkpoint validation**:
   ```bash
   cd ..
   ./workshop_check.sh 1
   ```

3. **Follow the instructions** from the validation script

## Folder Structure

After completing Checkpoint 1, your structure will be:

```
student_workspace/
├── venv/              # Virtual environment (created by script)
├── templates/         # HTML templates (created by script)
├── static/            # CSS, JS, images (created by script)
├── app.py             # Your Flask app (you create this)
└── .env               # Environment variables (optional)
```

## Checkpoints

### Checkpoint 1: Environment Setup
```bash
../workshop_check.sh 1
```
- Creates virtual environment
- Installs Flask
- Sets up project folders

### Checkpoint 2: First Flask App
```bash
../workshop_check.sh 2
```
- Validates your `app.py`
- Checks Flask imports
- Tests if app runs

### Checkpoint 3: Tailwind CSS
```bash
../workshop_check.sh 3
```
- Checks HTML templates
- Validates Tailwind integration
- Verifies utility classes

### Checkpoint 4: AI Integration
```bash
../workshop_check.sh 4
```
- Checks environment variables
- Validates API endpoints
- Tests full integration

## Tips

- ✅ Always activate virtual environment before working
- ✅ Run validation after each major change
- ✅ Read error messages carefully
- ✅ Ask instructor if stuck
